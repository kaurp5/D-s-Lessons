# D's Lessons

A gentle, modern lesson diary site built for GitHub Pages.

## What is included
- Stylish landing page
- Lesson archive
- First example lesson
- Decap CMS admin area at `/admin/`

## To publish with GitHub Pages
1. Open the repository settings.
2. Go to **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select your default branch and the `/ (root)` folder.
5. Save.

## Important Decap CMS note
To use Decap CMS with GitHub auth, you will typically need a compatible authentication flow such as Netlify Identity or another supported Git Gateway/auth setup. The site itself works as a static website immediately, but CMS editing may require one extra auth step depending on how you want to manage content.
